CSV Format Accepted

Your CSV file can use these column names:

question, option_1/option1/opt1/A, option_2/option2/opt2/B, option_3/option3/opt3/C, option_4/option4/opt4/D, answer Example: question,option_1,option_2,option_3,option_4,answer "What is 2+2?","3","4","5","6",option_2

JSON Format Accepted

Your JSON file should follow this structure:

{ "exam": "Networking Systems II", "subject": "Computer Science", "class": "SS 3", "duration": 30, "questions": [ { "question": "A networking system is best described as", "A": "a) a single standalone computer", "B": "b) interconnected devices sharing data and resources", "C": "c) a software application", "D": "d) an operating system", "answer": "B" } ] }

The "class" field is optional. If omitted, the exam will be available to all classes.
